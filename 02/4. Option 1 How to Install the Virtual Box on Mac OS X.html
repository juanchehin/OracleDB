Hi Dear Friends,

You can skip this lesson in case you have another operating system different than the Mac OS X.

1) Download the two software packages that you need:

  1. Visit http://www.virtualbox.org/wiki/Downloads

  2. Click on the link for OS X hosts.

  3. This will bring up a download window with a filename similar to “VirtualBox-4.3.12-93733-OSX.dmg”.

  4. Save the file someplace where you will be able to find it later.

  5. The file is fairly small; only about 120MB on my computer.

  6. Next, click on the “All supported platforms” link next to  “VirtualBox 4.3.12 Oracle VM VirtualBox Extension Pack” and save that file to the same location.

2) Confirm that you have the two packages that you need:

Once you are done, check-in the Finder and confirm that you have the two packages that you need. On my Mac, the display looks something like this. Of course, yours may look somewhat different depending on the way you have set Finder to display files.

3) Mount the VirtualBox disk image to start the installation process

4) Install the VirtualBox program

  1. Double-click on the VirtualBox.pkg file to launch the actual installation.

  2. Click Continue as prompted and then click “Install.” You will probably be prompted for your OSX administrative account password to enable the installation to proceed.

  3. Enter the password to continue.

  4. Click on “Install Software.”

  5. On my MacBook Air, this took about one minute and I received a message saying that “The installation was successful.”

  6. Click Close.

✮Congrats!✮ Start the application and install the extension pack✮

  1. Switch into your Applications folder in the Finder.

  2. Locate VirtualBox and double-click on the application.

  3. It should launch and, with luck, you won’t see any errors.

  4. The very first thing you need to do is to install the extension pack. Without the extension, you won’t have access to a number of useful features.

  5. Click on VirtualBox > Preferences (or use the keyboard shortcut Cmd-,). Then click on “Extensions” in the top bar (#1 above).

  6. Now, click on the blue square with the yellow dropdown symbol (#2 above)

  7. This will bring up a file navigator – navigate to the location where you saved the extension pack and select it.

  8. You’ll see a warning message. Read it and click Install.

  9. You’ll have to read (or at least scroll through) a long Oracle license agreement before you can click OK or similar.

  10. You’ll be prompted for your local administrator password. Enter it and click to proceed.

  11. The installation will complete.

  12. You should be able to click “OK” to leave the VirtualBox preferences screen.
